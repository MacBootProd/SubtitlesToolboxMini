# Subtitles Toolbox mini
A mini toolbox to fix timing issues and more in .srt subtitles

Functions:
- Shift
- Re-sync
- Change Framerate
- Append 2 files
- Repair index
- Merge 2 files

Convert file format:
- ASS to SRT
- SUB to SRT
- TXT to SRT
- VTT to SRT
- SRT to plain text

Clean file from:
- Style
- Metadata
- SDH
- Lyrics


Download Subtitles Toolbox mini installers for Windows & macOS:
https://github.com/MacBootProd/SubtitlesToolboxMini/releases
